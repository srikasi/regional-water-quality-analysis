# Regional Water Quality Trend Analysis (QLD Open Data)

An environmental data engineering and statistical pipeline designed to ingest, clean, and analyze high-frequency water quality monitoring data from open-source Queensland government datasets.

## Project Overview
This project applies modern data analytics workflows to environmental and telemetry-style datasets. Drawing on a professional background managing large-scale industrial water systems, this pipeline automates the processing of historical water quality arrays to identify structural pollution trends, seasonal variances, and regulatory compliance risks.

*   **Tools and Libraries:** Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook
*   **Data Architecture Platform:** Open-source public infrastructure data integration
*   **Target Data Source:** Queensland Government Open Data portal (Water Monitoring Systems)

---

## Planned Technical Pipeline & Workflow
*   **Environmental Data Ingestion:** Automated downloading and loading of localized public water quality CSV matrices.
*   **Telemetry Stream Alignment:** Standardizing time-series monitoring intervals and handling missing or out-of-bounds sensor values.
*   **File-System Consolidation:** Merging separate regional monitoring location datasets across unified chronological keys into a structured DataFrame.
*   **Statistical Trend Evaluation:** Utilizing rolling averages and variance testing to track specific parameter shifts (e.g., pH, turbidity, dissolved oxygen) over time.
*   **BI Visualisation:** Generating clear, publication-grade statistical plots mapping seasonal environmental fluctuations against safety limits.
